# Debian-and-Networks
В данном репозитории будут расположены конфиги и скрипты для настройки и небольшой автоматизации. 
Основной фронт работ будет выполнен для ОС Debian 12 и в перспективе будут рассмотрены оболочки сетевых устройств для понимания сетей и другие дистрибутивы Linux. Работать буду в PNETLAB, VMware и на свойм ствендовом ноутбуке ASUS x553m с Debian 12 на борту с кучей полезных и по своему интерсных вещей.
