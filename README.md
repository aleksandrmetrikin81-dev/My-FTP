# Debian-and-Networks
В данном репозитории будут расположены конфиги и скрипты для настройки и небольшой автоматизации. 
Основной фронт работ будет выполнен для ОС Debian 12. Я в основном взаимодействую с VMware и со своим ствендовом ноутбуке ASUS x553m с Debian 12 на борту с кучей полезных и по своему интерсных вещей (смотри в папку configs).
