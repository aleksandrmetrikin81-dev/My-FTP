В данном репозитории расположены основные конфигурационные файлы моего сервера FTP. 
Данный проект реализован на ноутбуке ASUS x553m с Debian 12.
Основное взаимодействие с ним осуществляется через файловый менеджер Filezilla и через командную строку Putty (конечно только в случаях администрирования)
