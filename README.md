# Debian-and-Networks
В данном репозитории будут расположены конфиги и скрипты для настройки и небольшой автоматизации. 
Основной фронт работ будет выполнен для ОС Debian 12 и в перспективе будут рассмотрены оболочки сетевых устройств для понимания сетей и другие дистрибутивы Linux. Планирую активно работать в PNETLAB, а пока я в основном взаимодействую с VMware и со своим ствендовом ноутбуке ASUS x553m с Debian 12 на борту с кучей полезных и по своему интерсных вещей.
